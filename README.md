# bekzhanno_coingecko
## Title:
Assignment_1 CoinGecko
## Installation

PyPI
```bash 
pip install pycoingecko
```
or from source
```bash
git clone https://github.com/man-c/pycoingecko.git
cd pycoingecko
python3 setup.py install
```
## Usage
```bash
from pycoingecko import CoinGeckoAPI
cg = CoinGeckoAPI()
```
